Регистрация организации
Инструкции Партнерская программа Регистрация организации

Подсистема позволяет указать основные данные об организации, торговых точках, сотрудниках и настроить общую работу системы.

Для большего удобства настройки разделены по нескольким модулям:

  • Организации;
  • Подразделения;
  • Сотрудники;
  • Дополнительно;
  • Справочник прав.

Для существующих организаций будут отображены следующие данные:

  • ID - внутренний идентификатор организации в системе;
  • Статус - позволяет определить статус организации (Активен/Неактивен);
  • ИНН;
  • Дополнительно;
  • Справочник прав.
